# Credilix Acessos - Frontend

Interface web do sistema de acessos Credilix.

## Rodar local

1. Copie `.env.example` para `.env`.
2. Garanta que a API backend está rodando em `http://127.0.0.1:5050` (ou ajuste `VITE_API_BASE_URL`).
3. Rode:
   - `npm install`
   - `npm run dev`

## Variáveis

- `VITE_API_BASE_URL`: base da API backend.

## Módulos do painel

- Login
- Dashboard
- Usuários
- Produtos
- Tabelas de comissão
- Conteúdos
